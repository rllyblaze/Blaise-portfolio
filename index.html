Cloud-Based Language Translation Service

Cloud-Based Language Translation Service

This project is a serverless language translation service built using AWS. It allows authenticated users to submit text for translation and track the status of their request. The backend is powered by AWS Lambda, API Gateway, and Dynamo DB, while Amazon Translate handles text translation.

User authentication is managed with Amazon Cognito, ensuring secure access. The entire infrastructure is deployed using AWS Serveless Appliation Model (SAM).
